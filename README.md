# Multiplayer Wordle-Inspired Game

A **multiplayer "guess the word" game** inspired by the *New York Times' Wordle*, made with **Godot 4** using the **High-Level Multiplayer API**.  
This project uses **[@SlayHorizon](https://github.com/SlayHorizon) [godot-tiny-mmo](https://github.com/SlayHorizon/godot-tiny-mmo) server-client one project architecture**.
